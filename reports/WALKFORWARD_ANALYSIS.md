## 三、过拟合检测指标

### 3.1 DSR (Deflated Sharpe Ratio)

```python
DSR = Φ((SR - E[SR_max]) / σ)
```

- DSR > 0.95: 策略有效
- DSR < 0.95: 可能过拟合

### 3.2 IC 衰减检测

```
IC衰减比 = OOS_IC / IS_IC

正常: > 0.60
警告: 0.40 - 0.60
危险: < 0.40
```
